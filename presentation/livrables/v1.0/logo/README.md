### **Typographie**

- **Police** : "Nunito" (Google Fonts)
- **Source** : [Google Fonts - Nunito](https://fonts.google.com/specimen/Nunito)
- **Justification** :
  - Forme arrondie, friendly et moderne
  - Parfait pour l'esprit "robot compagnon doux"
  - Lisible à petite taille (favicon)
  - Distinctive sans être agressive
  - Open source, utilisable partout

**Style typographique** :
- Poids : Bold (700) pour "BBIA"
- Tracking : 0.05em (léger espacement)
- Hauteur ligne : 1.2
- Rendering : Optimisé pour écran (subpixel)

## 💡 Justification Créative

### **Choix de Conception**

**Mascotte Robot** :
- **Forme arrondie et douce** : Évite l'agressivité, transmet gentillesse et accessibilité
- **Yeux grands et expressifs** : Permettent de transmettre des émotions (curiosité, bienveillance)
- **Lumière centrale (cœur/cristal)** : Symbolise l'IA, la "source" de BBIA, lien avec réseau neuronique
- **Antennes subtiles** : Référence discrète au robot Reachy Mini, sans copier directement
- **Posture ouverte** : Invite à l'interaction, posture d'accueil

**Palette couleurs** :
- **Bleu principal (#87bcfa)** : Couleur principale palette, douceur tech, rassurante
- **Violet (#A680FF)** : Accent créatif, lien avec réseau lunaire, innovation
- **Turquoise (#60e9e1)** : Halo lumineux, symbolique lunaire Arkalia, éthéré
- **Justification globale** : Palette cohérente avec brief, accessible, jamais agressive

**Typographie "BBIA"** :
- **Nunito Bold** : Arrondi, friendly, moderne
- **Placement** : À droite de la mascotte (composition horizontale)
- **Taille relative** : Harmonieuse avec mascotte (ratio 1:1.2)
- **Justification** : Lisible, distinctive, cohérente avec esprit "douceur tech"

**Lien avec ADN BBIA** :
- ✅ Robot compagnon doux : Mascotte arrondie, yeux expressifs
- ✅ IA bienveillante : Lumière centrale, posture ouverte
- ✅ Symbolique lunaire Arkalia : Couleur turquoise, effet éthéré
- ✅ Futur tech doux : Palette douce, jamais agressif
- ✅ Accessibilité : Contraste suffisant, lisible petite taille

### **Variantes Proposées**

**Variante A - Logo Complet (Recommandée)** :
- Mascotte + Typographie "BBIA"
- Usage : Header web, documentation, communication principale
- Composition : Horizontale, ratio 3:1
- Différences : Version standard, tous éléments visibles

**Variante B - Mascotte Seule** :
- Uniquement la mascotte (sans texte)
- Usage : Favicon, avatar, espace restreint
- Composition : Carrée, ratio 1:1
- Différences : Pas de typographie, focus sur mascotte

**Variante C - Typographie Seule** :
- Uniquement "BBIA" stylisé
- Usage : Espace très restreint, version texte-only
- Composition : Horizontale
- Différences : Pas de mascotte, typographie plus grande

**Variante D - Monochrome** :
- Version noir (fond clair) ou blanc (fond sombre)
- Usage : Impression, contraintes couleurs
- Composition : Identique variante A/B
- Différences : Pas de couleurs, niveaux de gris seulement

### **Alternatives Créatives Explorées**

**Exploration 1 - Mascotte plus géométrique** :
- Testé : Formes plus angulaires, style plus "tech"
- Résultat : Trop froid, perd l'esprit "compagnon doux"
- **Justification rejet** : Non conforme à l'ADN BBIA "douceur tech"

**Exploration 2 - Palette plus saturée** :
- Testé : Couleurs plus vives, néon plus fort
- Résultat : Trop agressif, perd la douceur requise
- **Justification rejet** : Brief demande "douceur, non agressif"

**Exploration 3 - Typographie script** :
- Testé : Police cursive, plus organique
- Résultat : Moins lisible petite taille, moins moderne
- **Justification rejet** : Priorité lisibilité et modernité

## 📦 Utilisation

### **Pour Développeurs**

1. **Copier les fichiers SVG** dans `assets/logo/` ou `static/img/logo/`
2. **Utiliser le SVG** pour qualité vectorielle (scalable, léger)
3. **Générer PNG si nécessaire** (voir `exports/README_EXPORTS.md`)
4. **Référencer dans code** via chemin relatif :
   ```python
   LOGO_PATH = "assets/logo/logo_bbia_horizontal.svg"
   ```

**Exemple Python** :
```python
# Dans application web
from pathlib import Path

LOGO_DIR = Path(__file__).parent / "assets" / "logo"
logo_path = LOGO_DIR / "logo_bbia_horizontal.svg"
```

### **Pour Documentation**

**Markdown** :
```markdown
![BBIA Logo](logo_bbia_complet.png)

Le logo BBIA représente...
```

**Sphinx/RST** :
```rst
.. image:: logo_bbia_complet.png
   :alt: BBIA Logo
   :width: 300px
```

### **Pour Communication**

**Email signature** :
```html
<img src="logo_bbia_horizontal.svg" alt="BBIA" width="150">
```

**Présentations** :
- Utiliser PNG haute résolution (2048x2048px)
- Format recommandé : Logo horizontal pour slides

**Réseaux sociaux** :
- Profil : Mascotte seule (512x512px)
- Header : Logo horizontal (1920x1080px si adapté)
